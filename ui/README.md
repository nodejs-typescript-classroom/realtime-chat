# realtime chat ui

This repository is for chat system web ui part

## init script

```shell
pnpm create next-app ui
```

## install need package

```shell
cd ui
pnpm add socket.io-client
pnpm add -D @types/ws
```

## install shadcn-ui

```shell
npx shadcn-ui@latest init
```

## init with ui

```shell
npx shadcn-ui@latest add textarea button form
```
